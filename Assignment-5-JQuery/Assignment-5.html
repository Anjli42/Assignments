1. What is jQuery?

  • jQuery is a lightweight, "write less, do more", JavaScript library.

  • The jQuery library contains the following features:

    • HTML/DOM manipulation
    • CSS manipulation
    • HTML event methods
    • Effects and animations


2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

  • How to Apply CSS Using JQuery :

    • The css() method sets one or more style properties for the selected elements.

    • Syntax : $("selector").css({"property-1": "value-1", "property-2": "value-2"});

    • Example :
      <script>
      $(document).ready(function(){
      $("button").click(function(){
      $("p").css({"background-color": "yellow", "font-size": "200%"});
      });
      });
      </script>

  • How to Add Class and Remove Class in Jquery

    • addClass() - Adds one or more classes to the selected elements.

    • removeClass() - Removes one or more classes from the selected elements.

    • toggleClass() - Toggles between adding/removing classes from the selected elements.

  • JQuery Animation :

    JQuery Effects Syntax Description
    hide() $(selector).hide(speed, callback); Hide elements
    show() $(selector).show(speed, callback); Show elements
    toggle() $(selector).toggle(speed, callback); Display hidden element or hide visible element
    fadeIn() $(selector).fadeIn(speed, callback); Fade in a hidden element.
    fadeOut() $(selector).fadeOut(speed, callback); Fade in a visible element.
    fadeTo() $(selector).fadeTo(speed, callback); Fading to a given opacity (value between 0 and 1)
    fadeToggle() $(selector).fadeToggle(speed, callback); Display or hide the specified element by animating their opacity
    slideDown() $(selector).slideDown(speed, callback); Slide down an element
    slideUp() $(selector).slideUp(speed, callback); Slide up an element
    slideToggle() $(selector).slideToggle(speed, callback); Toggle slide
    animate() $(selector).animate(speed, callback); Create custom animations
